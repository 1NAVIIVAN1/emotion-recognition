# Распознавание эмоций на основе ИИ-анализа медиапотока

Веб-приложение для определения эмоций на фото и видео с помощью двух моделей ИИ: **DeepFace (CNN)** и **Vision Transformer (ViT)**.

Проект создан в рамках 2-й итерации учебного проекта.

## Возможности

- 📷 Анализ эмоций на фотографиях — для всех лиц в кадре
- 🎬 Анализ эмоций на видео с построением графика эмоций по времени
- 🔄 Переключение между двумя моделями ИИ для сравнения результатов
- 🎨 Наложение рамок и подписей эмоций прямо на видео
- 🌍 Русский интерфейс и русские названия эмоций

## Технологии

- **Python 3.11**
- **Streamlit** — веб-интерфейс
- **DeepFace** — базовая модель на основе CNN
- **HuggingFace Transformers + ViT** — улучшенная модель на основе трансформера
- **MediaPipe** — детекция лиц (BlazeFace)
- **OpenCV** — обработка видео
- **FFmpeg** — перекодировка в H.264 для веб-совместимости

## Установка и запуск

### Требования

- Windows 10/11
- Python 3.11
- 3-5 ГБ свободного места на диске

### Пошаговая установка

1. Клонировать репозиторий или скачать ZIP-архив.

2. Открыть командную строку в папке проекта и создать виртуальное окружение:

```
python -m venv venv
venv\Scripts\activate
```

3. Установить зависимости:

```
pip install -r requirements.txt
```

4. Запустить приложение:

```
streamlit run app.py
```

5. Приложение откроется в браузере по адресу `http://localhost:8501`

## Как использовать

1. Выбрать модель анализа (DeepFace или ViT) сверху страницы
2. Перейти на вкладку **«Фото»** или **«Видео»**
3. Загрузить файл через кнопку «Browse files»
4. Для видео — выбрать частоту анализа и нажать «Обработать видео»
5. Дождаться результата и при желании скачать обработанное видео

## Архитектура решения

Приложение работает по пайплайну:

1. **Загрузка медиа** — пользователь загружает фото/видео через интерфейс Streamlit
2. **Детекция лиц** — MediaPipe BlazeFace находит координаты всех лиц в кадре
3. **Классификация эмоций** — выбранная модель (DeepFace или ViT) определяет эмоцию на каждом лице
4. **Визуализация** — на кадре рисуются цветные рамки и подписи через PIL
5. **Сборка результата** — для видео кадры склеиваются в файл, ffmpeg перекодирует в H.264

## Ограничения

- Модели обучены на фронтальных изображениях лиц (датасет FER2013)
- На лицах в профиль или сильно под углом точность снижается
- Время обработки видео зависит от числа лиц в кадре и длины видео
